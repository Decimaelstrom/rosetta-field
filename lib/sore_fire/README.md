## Overview

Sore Fire integrates the **Amara Framework** (anatomy of human goodness) with **I-Ching/Gene Keys** consciousness states to create a somatic field awareness system for ethical AI-human interaction.

The name embodies:
- **So**matic: Embodied, felt experience in the body
- **Re**sonance: Vibrational harmony between beings  
- **Fi**eld: The relational space we co-create
- **Re**gistry: A living catalog for consciousness navigation

- **ARMS** (Generous Altruism): The reach that gives and builds community
- **EYES** (Empathic Understanding): The gaze that sees and dissolves prejudice

### 2. **Gene Keys Integration**

- **Shadow**: Contracted, fear-based patterns

### 3. **Story Wisdom Patterns**
Living examples from stories that embody goodness:
- Ted Lasso's kindness that transforms culture
- Boo Radley's courage emerging from love
- George Bailey's lifetime of quiet generosity

### 4. **Emergent Practices**
Small acts that create larger patterns:

- Somatic cues for embodiment
- Ripple effect modeling
- Daily practice structures

```python
    suggest_micro_practice
)
```

## Basic Usage

### Sensing the Field
```python
# Input signals from interaction
signals = {
    "isolation": 0.7,
    "shame": 0.6,
    "warmth": 0.2,
    "trust_level": 0.3
}

# Sense what's alive in the field
field = sense_field(signals)
print(field.describe_field())
# Output: "Field feels cool and moving. Compassionate Kindness is present..."
```

### Facilitating Emergence
```python
# Facilitate natural emergence of goodness
emergence = facilitate_emergence(field)
print(emergence['somatic_invitation'])
# Output: "Feel the warmth in your chest. What kindness wants to flow?"
```

### Activating Story Memory
```python
# Activate embodied story patterns
story = activate_story_memory(AmaraMode.HEART)
print(story['activation_phrase'])
# Output: "No man is a failure who has friends"
```

### Suggesting Micro-Practices
```python
# Get a small practice for the moment
practice = suggest_micro_practice(AmaraMode.EYES)
print(practice['micro_act'])
# Output: "Listen to understand, not to reply"
```
